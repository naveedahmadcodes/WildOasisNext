#  The Wild Oasis

A modern full-stack hotel management application built with **Next.js**. The Wild Oasis enables hotel staff to manage cabins, bookings, guests, and reservations through a clean, responsive, and user-friendly dashboard.

## Features

- Secure Authentication
- Cabin Management
- Booking & Reservation System
- Guest Management
- Dashboard with Booking Statistics
- Fully Responsive Design
- Fast Server-Side Rendering with Next.js
- Modern & Clean UI

## Tech Stack

- Next.js
- React.js
- JavaScript
- Supabase
- Tailwind CSS
- NextAuth (Authentication)

## Installation

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
